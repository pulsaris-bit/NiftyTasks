# NiftyTasks

NiftyTasks is een moderne, minimalistische taakbeheer-applicatie ontworpen voor eenvoud en stijl. Het helpt je om je dagelijkse taken te organiseren met een focus op een rustige gebruikersinterface en vloeiende interacties.

## 🚀 Over de App

NiftyTasks biedt een intuïtieve manier om taken te beheren:
- **Taakbeheer**: Eenvoudig toevoegen, voltooien en verwijderen van taken.
- **Projecten & Labels**: Organiseer taken in verschillende categorieën met gepersonaliseerde kleuren.
- **Prioriteiten**: Geef aan welke taken direct aandacht nodig hebben.
- **Minimalistische UI**: Een overzichtelijk dashboard met een inklapbare zijbalk voor maximale focus.
- **Responsive Design**: Perfect bruikbaar op zowel desktop als mobiele apparaten.

## 🛠 Techniek

De applicatie is gebouwd met moderne webtechnologieën voor een snelle en robuuste ervaring:

- **Frontend Framework**: [React 19](https://react.dev/) met [Vite](https://vitejs.dev/) voor een razendsnelle ontwikkelervaring.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) voor een modern en flexibel design.
- **Animaties**: [Framer Motion](https://www.framer.com/motion/) voor vloeiende overgangen en interface-interacties.
- **Icons**: [Lucide React](https://lucide.dev/) voor een consistente en duidelijke iconografie.
- **Full-stack**: Gebruikt een [Express](https://expressjs.com/) server voor API-afhandeling en statische hosting.
- **Data**: Lokale opslag (SQLite via `better-sqlite3` op de server of LocalStorage voor client-side persistence).

## 📦 Installatie

Volg deze stappen om het project lokaal te draaien:

1. **Clone de repository**:
   ```bash
   git clone <repository-url>
   cd niftytasks
   ```

2. **Installeer afhankelijkheden**:
   ```bash
   npm install
   ```

3. **Start de development server**:
   ```bash
   npm run dev
   ```
   De applicatie is nu bereikbaar op `http://localhost:3000`.

4. **Bouwen voor productie**:
   ```bash
   npm run build
   ```
